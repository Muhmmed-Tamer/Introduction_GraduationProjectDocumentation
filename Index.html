Overview

The AuthController is a RESTful WebAPI controller that handles user authentication operations. It provides secure login functionality using JWT tokens and follows clean architecture principles with CQRS pattern implementation.

Base URL: http://introductiongraduationprojectapi.runasp.net/
Content-Type: application/json
Framework: ASP.NET Core 8.0
Authentication: JWT Bearer Token

Architecture

The authentication system follows a clean architecture with separation of concerns, ensuring maintainability and testability. The CQRS pattern separates read and write operations for better performance and scalability.

API Endpoints

POST /api/Auth/Login
Authenticates a user with email and password credentials and returns JWT tokens for authorized access to protected resources.
